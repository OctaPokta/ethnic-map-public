# Middle East Ethnic Map

## Project Overview
MMapa is an interactive, high-resolution mapping application designed to visualize ethnic distributions across the Middle East. The application provides detailed demographic data, cinematic audio feedback, and curated dossiers for major capital cities.

## Key Features
* **High-Resolution Background**: Utilizes a detailed 6194 by 3876 pixel geographic base map.
* **Coordinate-Based Accuracy**: City pins are mathematically mapped to raw image pixels, ensuring they remain geographically locked across all device sizes.
* **Dynamic Demographics**: Interactive donut charts and bar graphs display ethnic composition for selected countries.
* **Responsive Design**: Fully adapted for mobile and desktop, including multi-touch support for pinch-to-zoom and panning on mobile devices.
* **Themed Interface**: Includes a night mode toggle and a collapsible sidebar to maximize map visibility on smaller screens.
